## 📌 Overview

A browser-based **Tic-Tac-Toe (XO) game** built entirely with HTML, CSS, and JavaScript. Two players take turns on a 3×3 grid — first to get three in a row wins!

---

## 🎮 How to Play

1. Player **X** goes first — click any empty cell
2. Player **O** takes their turn
3. First player to align **3 symbols** (row, column, or diagonal) wins
4. If all cells are filled with no winner → **Draw!**
5. Click **Restart** to play again

---

## ✨ Features

- 🎯 **Two-Player Mode** — play locally with a friend
- 🏆 **Win Detection** — all 8 winning combinations checked
- 🤝 **Draw Detection** — handles tied game state
- 🔄 **Restart Button** — reset the board instantly
- 📱 **Responsive Design** — works on all screen sizes
- 🎨 **Visual Feedback** — highlighted winning cells

---

## 🧠 Game Logic

```
Win conditions checked after every move:
  Rows:      [0,1,2] [3,4,5] [6,7,8]
  Columns:   [0,3,6] [1,4,7] [2,5,8]
  Diagonals: [0,4,8] [2,4,6]
```
